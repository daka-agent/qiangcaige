# 抢菜歌 🥬🎵

> 一首7岁男孩小卡写给老爸的童趣儿歌

---

## 关于这首歌

小卡（7岁）某天随手写在便利贴上的一首《抢菜歌》，歌词充满了孩子气的幽默和天真——

"抢啊吃啊爸，老爸抢菜不是慢而是快……抢到9年还在抢菜，吃撑还在抢菜……爸爸是抢菜之神！"

在小卡眼里，爸爸饭桌上抢菜又快又猛，是真正的 **"抢菜之神"**。

## 作品特点

- 🎵 **Suno AI 儿歌演唱版**：童声欢快演唱，朗朗上口
- 🎨 **手稿原作展示**：小卡手写便利贴原作完整保留
- 📝 **同步歌词滚动**：歌词逐句高亮，跟着唱
- 📱 **单文件 H5**：一个 `index.html` 搞定，浏览器即开即用
